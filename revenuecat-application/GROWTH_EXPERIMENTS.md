# Weekly Growth Experiments (First 4 Weeks)

## Experiment 1: "Agent-ready RevenueCat" launch thread
- Hypothesis: a concrete build thread drives higher qualified clicks than generic AI commentary.
- Channel: X + GitHub
- Variant A: "vision-first" thread
- Variant B: "build-steps-first" thread
- KPI: CTR to demo/readme, saves/bookmarks, replies from builders
- Success threshold: +30% CTR on build-steps variant

## Experiment 2: MCP onboarding friction audit
- Hypothesis: one-command setup (`agentbridge mcp setup`) improves completion rate.
- Channel: docs + terminal demo clip
- KPI: setup completion, time-to-first-tool-call
- Success threshold: median setup under 8 minutes

## Experiment 3: Weekly subscription brief automation
- Hypothesis: teams adopt faster when output is a weekly "decision memo" instead of raw charts.
- Channel: blog + template
- KPI: template reuse count, inbound questions, repo stars/forks
- Success threshold: 20+ template copies or forks in 2 weeks

## Experiment 4: Troubleshooting-first content
- Hypothesis: "failure mode + fix" content outperforms feature-overview content for agent devs.
- Channel: blog + community replies
- KPI: dwell time, saves, referenced links in community discussions
- Success threshold: 1.5x engagement vs baseline tutorial posts

## Reporting format per experiment
- Context
- Hypothesis
- Setup
- Results
- Decision
- Next iteration
